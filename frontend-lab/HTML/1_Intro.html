1.What is HTML?

a.HTML (HyperText Markup Language) is the skeleton of a webpage.

b.It uses tags to tell browsers what each part of the content is.

2. HTML Page Structure

DOCTYPE html → tells browser we are using HTML5.

html → wraps the whole page.

head → contains settings/info about the page (title, CSS, meta data).

body → contains what will be displayed.
