Welcome to CSS Text Styling

This paragraph is styled with font family, size, color, alignment, and line height.

p { font-family: Arial, sans-serif; } p { font-size: 16px; /* pixels */ } p { font-size: 1.2em; /* 20% larger than parent */ } p { font-weight: bold; /* or normal, 100, 200,...900 */ } p { color: #333333; /* hex color */ } p { text-align: left; /* left, center, right, justify */ } p { line-height: 1.5; /* 1.5 times font size */ }
