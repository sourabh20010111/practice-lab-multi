Overflow Examples

overflow: visible (default) — content spills out.

overflow: hidden — extra content is clipped.

This text is clipped when it goes beyond the box. You can't scroll to see it. Lorem ipsum dolor sit amet, consectetur adipiscing elit.

overflow: scroll — scrollbars always visible.

This box always has scrollbars, even if not needed. Lorem ipsum dolor sit amet, consectetur adipiscing elit. Proin ut ipsum eget velit dictum euismod.

overflow: auto — scrollbars appear only if needed.

This box only shows scrollbars when content overflows. Lorem ipsum dolor sit amet, consectetur adipiscing elit. Proin ut ipsum eget velit dictum euismod.
