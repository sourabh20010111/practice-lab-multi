CSS Positioning — Step by Step

Follow the steps below. Read the label, then observe how each positioning mode behaves.

Step 1 — Static (default)

Elements are in normal document flow. No manual offsets apply.

Step 4 — Fixed (relative to viewport)

The yellow header below is position: sticky. It behaves like relative until it reaches the top edge, then it sticks there while this section scrolls past.
